# B_TEA Frontend

React-based frontend for the B_TEA Intelligent Expense Analysis Platform.

## Setup

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Tech Stack
- React 18+
- TypeScript
- TailwindCSS
- Recharts
- Axios

- File upload interface
- Analytics dashboard
- Charts and visualizations
